# Tags

Organize and filter your API keys with custom labels.

## Key Features

- Add custom tags to any API key directly from the card
- Filter the keys list by tags using the top filter bar
- Tags are searchable — type a tag name in the search bar
- Tags are stored per app and persist across sessions

## Details

Tags let you categorize API keys by team, environment, use case, or any label you choose. Click the **+** button next to an app name to add tags. Tags can be removed by clicking the **x** on each tag pill. Use the **Tag filter** dropdown in the top bar to narrow down the list. Each tag can be up to 30 characters.
