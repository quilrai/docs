# Red Team Testing

Test your guardrails configuration against adversarial prompts.

## Overview

Red team testing sends a battery of adversarial prompts through your gateway to verify that guardrails are catching threats as expected. Use it after changing guardrail configurations to validate that your setup is working correctly.

## Key Features

- Run automated adversarial tests against any API key
- Tests cover prompt injection, jailbreak, and data extraction
- View test results and history per app

## How It Works

1. Select an API key to test
2. Run the red team test suite
3. Review results showing which attacks were blocked, monitored, or passed through
4. Adjust guardrail configurations based on findings
5. Re-run to verify changes
